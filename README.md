# CivicScan — Urban Infrastructure Risk Detection

CivicScan is a full-stack smart city platform that allows citizens to report infrastructure issues and uses AI to analyze risk severity in real-time.

## Features

###  Smart Report System

* Upload infrastructure images
* AI auto-detects issue type & severity
* Location-based reporting with geolocation

### AI Risk Analyzer

* Generates risk score (1–10)
* Assigns priority (Critical / Moderate / Low)
* Provides repair recommendations

### Live City Map

* Real-time issue visualization
* Color-coded severity markers
* Auto-refresh every 5 seconds

### Dashboard

* Total reports
* Critical issues count
* Average risk score
* Estimated repair cost

###  Admin Panel

* Track issue status (Reported → In Review → Fixed)
* Update and manage infrastructure issues

---

##  Tech Stack

* Frontend: Next.js + Tailwind CSS
* Backend: Flask (Python)
* AI: OpenRouter API (Mistral + LLaMA Vision)
* Maps: Leaflet.js

---

##  Setup

1. Clone the repo
2. Add your API key:

```
OPENROUTER_API_KEY=your_key_here
```

3. Run frontend & backend
